Prescriptions

P1. Accessibility CRITICAL

What: Logout button is emoji-only. Sidebar nav has no semantic role. Hero stat updates silently. No skip-to-content.

Why: Airport security tool, BCAS audit likely includes accessibility. Keyboard-only and screen-reader users cannot operate.

Fix: /design interaction — add aria-labels, aria-live regions, skip-to-content link.

P2. Responsiveness WATCH

What: No mobile breakpoints below 1024px. Monthly pivot overflows on phones.

Why: CSO may open monthly report on iPad before board meeting.

Fix: /design responsive — add 768px and 480px breakpoints. Consider stacked card view for monthly on phones.

P3. Speed WATCH

What: exceljs and file-saver in main bundle despite only being needed on Export click.

Why: 150KB is meaningful on slow cargo-terminal connections.

Fix: Dynamic import of excel.ts inside the export handler.

P4. Readability WATCH

What: Light mode text contrast borderline. 12px text in tables.

Why: Aging workforce, 8-hour shifts. WCAG AAA target for body text.

Fix: /design recolor — deepen text-1 from #475569 to #334155.

Verdict

The interface has crossed from "looks generated" to "looks intentional." Safe to demo. Not safe to ship without fixing accessibility and running a shadow logging period.
