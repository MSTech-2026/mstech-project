Prescriptions

P1. Responsiveness WATCH

What: No breakpoints. Hero metric wraps uncontrolled on tablet. 9-column table needs horizontal scroll on narrow screens.

Why: Admins may check the dashboard on tablets during terminal walkthroughs.

Fix: /design responsive. Add breakpoints at 768px and 1024px.

P2. Speed WATCH

What: Google Fonts @import is render-blocking. No font-display: swap parameter.

Why: Slow connections at cargo terminals add 200-500ms blank screen.

Fix: Add &display=swap to the font URL. Or self-host with @font-face.

P3. Accessibility WATCH

What: No aria-live on dynamic content. No accessibilityLabel on mobile inputs. Error messages are red-text-only.

Why: Screen reader users and color-blind users cannot fully operate the interface.

Fix: /design interaction. Add aria-live, accessibilityLabel, and error message prefixes.
